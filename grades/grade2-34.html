Unidade: Escola de Comunicações e Artes - ( ECA )
Curso: Turismo - noturno
Duração
Ideal: 8 semestres        Mínima: 8 semestres        Máxima: 12 semestres

Informações específicas

1) OPTATIVAS ELETIVAS PARA INGRESSANTES ATÉ 2023: Os alunos deverão cursar 4 créditos em disciplinas optativas eletivas ao longo curso (para ingressantes até 2019 não considerar os créditos trabalho e para ingressantes a partir de 2020 considerar aula e trabalho). 

2) OPTATIVAS ELETIVAS PARA INGRESSANTES A PARTIR DE 2024: Os alunos deverão cursar 8 créditos em disciplinas optativas eletivas ao longo curso (considerar aula e trabalho). 

3) OPTATIVAS LIVRES PARA INGRESSANTES ATÉ 2024: Os alunos deverão cursar 12 créditos em disciplinas optativas livres ao longo curso (para ingressantes até 2019 não considerar os créditos trabalho e para ingressantes a partir de 2020 considerar aula e trabalho). 

4) OPTATIVAS LIVRES PARA INGRESSANTES A PARTIR DE 2025: Os alunos deverão cursar 8 créditos em disciplinas optativas livres ao longo do curso (considerar aula e trabalho) 

5) As disciplinas CRP0528 e CRP0556 são obrigatórias para ingressantes a partir de 2019. Os alunos que já cursaram CRP0528 até 2018, poderão considerar os créditos da disciplina como optativa eletiva. 

6) Todos os alunos estão desobrigados de cursar CRP0455. Conforme o Departamento, os alunos que cursaram como obrigatória até 2018, poderão considerar como optativa eletiva. 

7) As disciplinas CRP0474 (equivalente à CRP0562) e CRP0479 (equivalente à CRP0561 equivalente à CRP0350) são obrigatórias, também, para ingressantes de 2018. 

8) A disciplina CRP0458 é obrigatória, também, para ingressantes de 2016 e 2017. 

9) A disciplina CRP0488 (equivalente à CRP0563) é obrigatória, também, para ingressantes de 2016. 

10) Todos os alunos estão desobrigados de cursar as disciplinas CBD0100 e CRP0487. Os alunos que já cursaram CRP0487 até 2016, poderão considerar os créditos da disciplina como optativa eletiva.

11) A disciplina CCA0290 é obrigatória para ingressantes a partir de 2023.

12) A disciplina CCA0284 é obrigatória para ingressantes a partir de 2024.

13) Alunos que tenham a MPR8001 como obrigatória e ainda não cursaram, deverão cursar 8 créditos em optativas eletivas (120 horas). Alunos que já tenham cursado, deverão cumprir apenas 4 créditos em optativas eletivas (60 horas), conforme vinha sendo praticado. Ambos comtemplam hora aula e trabalho.

14) Os alunos ingressantes até 2022, que não cursaram CCA0203 e CCA0204, deverão cursar CCA0284 Mídia e Sociedade e CCA0290 Tecnologias da Comunicação na Sociedade Contemporânea, respectivamente, como equivalentes.

15) As 120 horas de AACs exigidas, são obrigatórias para alunos ingressantes a partir de 2022.

16) Horas de extensão obrigatórias para ingressantes a partir de 2023.

17) A disciplina CRP0560 é obrigatória para ingressantes a partir de 2022.

18) A disciplina CRP0555 é obrigatória para ingressantes até 2024 e optativa eletiva para ingressantes a partir de 2025.

19) A disciplina CRP0480 é obrigatória, também, para ingressantes de 2023.

20) A disciplina CRP0461 têm como equivalência, até 2024, a disciplina CRP0567, a partir de 2025 sua equivalência passou a ser CRP0569. Ingressantes até 2023 que cursaram CRP0461, podem cursar as disciplinas CRP0567 e CRP0569 como optativas livres. Para ingressantes a partir de 2024 as disciplinas CRP0567 e CRP0569 são obrigatórias.  

21) Para ingressantes até 2023, a disciplina CRP0567 não é requisito para CRP0569.


Legenda: CH = Carga horária Total; CE = Carga horária de Estágio; CP = Carga horária de Práticas como Componentes Curriculares; ATPA = Atividades Teórico-Práticas de Aprofundamento
